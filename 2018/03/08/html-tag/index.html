HTML标签属性

发表于 2018-03-08 | 更新于: 2018-03-11

html标签属性以键值对的形式出现

通用属性：

id: 元素的唯一标识
name: 元素的名称
class: 元素的类名
style: 元素行内样式

前三个往往在用css和js的时候才有效

常用标签属性：

align: 对齐方式
bgcolor: 背景色
border: 边框
